## 📁 Estructura del Proyecto

*   **/css**: Archivos de estilos para el diseño visual del sitio.
*   **/images**: Recursos gráficos y activos visuales.
*   **/js**: Scripts para la lógica y dinamismo del sitio.
*   **index.html**: Página principal del proyecto.
*   **pago.html**: Interfaz dedicada al proceso de pagos.
*   **construccion.html**: Página informativa de mantenimiento o desarrollo.
*   **gracias.html**: Página de confirmación o agradecimiento.

## 🚀 Cómo visualizar el proyecto

Para ver el proyecto en tu navegador:
1. Clona este repositorio en tu computadora.
2. Abre el archivo `index.html` directamente en tu navegador favorito.
